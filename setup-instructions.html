📸 Profile Image Setup

How to Add Your Profile Picture:

Step 1: Locate the profile image shared in the chat conversation
Step 2: Right-click on the image and select "Save image as..."
Step 3: Navigate to this folder:
c:\Users\user\Desktop\MEARN\portfo\images\
Step 4: Save the file with the exact name: profile.jpg
Step 5: Open index.html in your browser to view your portfolio
📌 Important Notes:
  • The image filename must be exactly profile.jpg
  • Make sure it's saved in the images folder
  • Recommended size: 400x400 pixels (square format)
  • Supported formats: JPG, PNG

📁 File Structure:

portfo/

├── styles.css
